# 1. 发货时效

## 1.1 现货商品
- 如果订单在每日16:00前完成支付且通过风控审核，则应在24小时内出库。
- 如果订单在每日16:00后完成支付，则应在48小时内出库。
- 如果遇到大促、极端天气或仓库系统维护，则发货时效可延长至72小时，订单页需展示对应提示。

## 1.2 预售商品
- 如果商品为预售，则以商品详情页标注的预计发货时间为准。
- 如果预售订单提前备货完成，则可提前发货，但客服不得承诺一定提前。

## 1.3 组合订单
- 如果同一订单包含多件商品，则默认整单发货。
- 如果部分商品缺货且系统支持拆单，则可拆单发货；拆单后每个子包裹独立显示物流信息。
- 如果订单包含会员权益类虚拟商品，则虚拟权益不影响实物发货时效。

# 2. 配送区域

## 2.1 支持配送区域
- 支持中国大陆大部分城市配送。
- 支持 App、小程序、官网下单。

## 2.2 不支持配送区域
- 不支持港澳台地区。
- 不支持西藏全域。
- 不支持青海玉树州、果洛州。
- 不支持新疆部分边远县域。
- 不支持海南三沙及其他需二次海运的离岛地址。
- 不支持军事管理区、保密单位、临时施工营地地址。
- 如果系统已限制下单，则用户无法提交订单。
- 如果系统未拦截但仓配复核发现不可达，则订单将被取消并原路退款。

# 3. 预计送达规则

## 3.1 标准预计时效
- 华东、华中、华北主要城市：出库后1至3个自然日送达。
- 华南、西南、东北主要城市：出库后3至5个自然日送达。
- 偏远可达地区：出库后5至8个自然日送达。
- 大件或需预约配送商品：以预约结果为准。

## 3.2 时效说明
- 预计送达时间为系统预测，不等于到货承诺。
- 如果遇到节假日、极端天气、交通管制、地址联系不上，则预计送达时间会顺延。
- 如果物流公司已完成派送但用户未签收，则不视为商家延迟发货。

# 4. 修改地址规则

## 4.1 可修改阶段
- 如果订单状态为“待支付”，则用户可自行取消后重新下单。
- 如果订单状态为“已支付待审核”或“待出库”，则用户可申请修改一次地址或联系人信息。
- 如果订单状态为“已出库未揽收”，客服可发起拦截改址申请，但不保证成功。

## 4.2 不可修改阶段
- 如果订单状态为“运输中”，原则上不支持修改收货地址。
- 如果物流商支持中转改址，则客服仅可协助提交申请，最终以物流商结果为准。
- 如果订单为智能门锁且已预约安装，则地址修改后原安装预约自动失效，需要重新预约。

## 4.3 改址失败
- 如果改址失败且用户不再需要商品，则需待包裹退回仓库后办理退款。
- 客服不得承诺“已经通知仓库就一定改成功”。

# 5. 物流异常处理

## 5.1 无物流轨迹
- 如果订单显示“已出库”超过24小时仍无物流轨迹，则客服应登记物流核查。
- 如果登记后48小时仍无轨迹更新，则可升级为异常工单。

## 5.2 长时间不更新
- 如果包裹在运输中超过72小时无轨迹更新，则客服应发起催查。
- 如果包裹在同一网点停留超过48小时，则客服应登记异常滞留。
- 如果用户急用，客服可在核实后优先升级人工处理，但不得直接承诺补发。

## 5.3 签收争议
- 如果系统显示已签收但用户表示未收到，则应先核对：
  1. 签收时间；
  2. 签收方式；
  3. 代收人信息；
  4. 收货地址周边前台、驿站、门卫情况。
- 如果核对后仍无法确认，则需创建丢件核查工单。

# 6. 丢件/破损处理

## 6.1 丢件处理
- 如果物流核查确认丢件，则用户可选择补发同款或退款。
- 如果同款无货，则仅支持退款或更换同价位在售同型号替代款，且需用户确认。
- 丢件确认目标时效为工单创建后5个工作日内。

## 6.2 外包装破损
- 如果用户签收前发现外包装明显破损、渗液、变形，则建议拒收并拍照留证。
- 如果用户已签收，则应在签收后24小时内提供外箱照片、面单照片、内件照片。
- 超过24小时反馈的外观破损，客服可登记，但不得直接认定为物流责任。

## 6.3 内件破损
- 如果商品开箱后发现主机裂损、配件折断、屏体碎裂，则按破损工单处理。
- 如果破损影响使用，则优先进入换货或退货判断。
- 如果仅外包装轻微压痕、不影响商品使用，则不直接视为破损赔付条件。

# 7. 大件商品配送

## 7.1 大件定义
- 单件商品毛重超过15kg，或最长边超过80cm，定义为大件商品。
- 智能净化器部分型号、扫地机器人套装部分型号可能归类为大件商品。

## 7.2 配送规则
- 大件商品可能由专线物流或预约配送完成。
- 如果配送区域无电梯且需人工搬运上楼，则最终以上门人员评估为准。
- 客服不得承诺“一定送货上门入户”。

## 7.3 签收建议
- 大件商品签收时，建议用户先检查外箱完整性再签收。
- 如果外箱严重破损，建议拒收并保留照片。
- 如果用户本人无法现场验货，可由代收人拍照留档。
