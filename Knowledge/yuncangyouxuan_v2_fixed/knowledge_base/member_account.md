# 1. 会员等级

## 1.1 等级划分
- 普通会员：成长值 0 至 999。
- 银卡会员：成长值 1000 至 2999。
- 金卡会员：成长值 3000 至 7999。
- 黑金会员：成长值 8000 及以上。

## 1.2 成长值获取
- 实物订单完成后，按商品实付金额1元=1成长值发放。
- 运费、退款金额、虚拟权益类商品不计入成长值。
- 如果订单发生全额退款，则对应成长值全部扣回。
- 如果订单发生部分退款，则按退款比例扣回成长值。

## 1.3 等级有效期
- 等级按最近365天累计成长值动态计算。
- 如果近365天成长值下降至较低等级区间，则会员等级自动调整。
- 等级调整以每日凌晨系统刷新结果为准。

# 2. 优惠券

## 2.1 优惠券类型
- 商品券：用于指定商品或指定品类。
- 运费券：仅抵扣配送运费。
- 会员券：仅限指定会员等级领取或使用。
- 补偿券：由客服工单发放，通常有更短有效期。

## 2.2 使用规则
- 同一订单最多可使用1张商品券和1张运费券。
- 优惠券不可兑现、不可转赠、不可拆分使用。
- 如果订单使用优惠券后退款，则已使用的优惠券是否返还以券规则为准。
- 如果券面明确写明“退款不返”，则退款后不返还。

## 2.3 有效期
- 普通领券后立即生效，过期时间以券面展示为准。
- 补偿券默认有效期为7个自然日，除非工单备注另有说明。
- 过期优惠券不支持补发或延期，除非系统故障有工单记录。

# 3. 积分

## 3.1 积分获取
- 实物订单完成后，按商品实付金额1元=1积分发放。
- 运费、虚拟权益、赠品不发放积分。
- 使用积分抵扣的部分不再重复获得积分。

## 3.2 积分使用
- 100积分可抵扣1元订单金额。
- 单笔订单积分抵扣上限为商品实付金额的20%。
- 积分不可兑换现金，不可转让，不可单独提现。

## 3.3 积分失效
- 积分自发放之日起至次年12月31日23:59有效。
- 如果订单全额退款，则对应积分全部扣回。
- 如果订单部分退款，则按退款比例扣回积分。
- 如果账户积分不足以扣回，账户将显示为“待抵扣积分”。

# 4. 发票

## 4.1 发票抬头管理
- 会员账户可保存最多5个常用发票抬头。
- 可保存个人抬头或企业抬头。
- 企业抬头需填写企业名称和纳税人识别号。
- 发票开具规则以 `payment_invoice.md` 为准。

## 4.2 发票记录
- 会员中心可查询近180个自然日内的电子发票申请记录。
- 如果订单已全额退款，则原发票记录会显示为“已作废”或“已失效待重开”。

# 5. 地址管理

## 5.1 地址数量
- 单个账号最多可保存20个收货地址。
- 可设置1个默认地址。
- 删除地址不影响历史订单。

## 5.2 地址校验
- 新增地址时，系统会校验是否在可配送范围内。
- 如果地址不在支持配送区域，则无法保存为有效地址。
- 如果从新设备登录后新增高风险地址，系统可能触发二次验证。

## 5.3 历史订单地址
- 历史订单地址仅用于订单履约和售后核验，不因用户修改地址簿而自动变更。
- 如果用户想修改已下单订单地址，需按 `logistics.md` 中的修改地址规则处理。

# 6. 账户安全

## 6.1 登录保护
- 同一账号30分钟内连续输错登录密码或验证码5次，将触发30分钟临时保护。
- 保护期间不可继续尝试敏感操作。
- 敏感操作包括修改登录手机号、修改收货人姓名、清空地址簿、申请高价值订单售后。

## 6.2 设备管理
- 同一账号最多保留5台常用设备登录记录。
- 新设备登录后，若短时间内连续修改地址、发票抬头、收货手机号，系统可触发风控审核。
- 风控审核期间，部分订单可能显示“已支付待审核”。

## 6.3 账户冻结
- 如果系统识别到异常登录、批量下单、异常退款申请等风险行为，账号可能被临时限制部分功能。
- 临时限制期间，用户仍可查看历史订单，但可能无法提交新订单或使用积分、优惠券。
- 客服不得直接承诺“马上解封”，需按人工复核流程处理。
