# 1. 商品分类

## 1.1 一级分类
- 家电
  - 智能净化器
  - 胶囊咖啡机
  - 扫地机器人
- 数码配件
  - 降噪耳机
  - 充电底座
  - 替换耳罩
- 智能家居
  - 智能门锁
  - 门锁网关
  - 门磁配件
- 会员权益
  - 云仓PLUS月卡
  - 云仓PLUS年卡

## 1.2 商品形态
- 整机类：可独立使用的主商品，例如智能净化器、扫地机器人。
- 配件类：用于替换或增强主商品的部件，例如滚刷、尘袋、耳罩、电源适配器。
- 耗材类：会随着使用消耗的商品，例如净化器滤芯、咖啡胶囊、扫地机尘袋。
- 会员权益类：虚拟权益，不涉及实物发货。

# 2. 重点商品说明

## 2.1 智能净化器
- 适用场景：卧室、书房、客厅等室内环境。
- 标准包装通常包含：主机、电源线、预装滤芯、说明书。
- 如果商品详情页写明“需自行安装滤芯”，则用户首次使用前需完成安装。
- 如果用户反馈“无法开机”，则先确认：
  1. 是否连接原装电源；
  2. 滤芯仓盖是否扣紧；
  3. 儿童锁是否开启。

## 2.2 降噪耳机
- 适用场景：通勤、办公、居家。
- 标准包装通常包含：耳机主体、充电线、收纳包、说明书。
- 如果用户反馈“降噪效果不明显”，则需先确认：
  1. 是否已开启主动降噪模式；
  2. 耳罩或耳塞是否贴合；
  3. 当前环境噪音是否为突发高频噪音。
- 客服不得承诺“任何环境都能完全静音”。

## 2.3 胶囊咖啡机
- 适用场景：家庭和小型办公场景。
- 标准包装通常包含：主机、水箱、接渣盒、说明书。
- 如果用户反馈“无法出咖啡”，则先确认：
  1. 水箱是否装满并卡紧；
  2. 是否使用适配型号胶囊；
  3. 是否完成首次引水；
  4. 是否提示除垢。

## 2.4 扫地机器人
- 适用场景：室内地面清洁。
- 标准包装通常包含：主机、充电座、边刷、抹布组件、说明书。
- 如果用户反馈“无法回充”，则先确认：
  1. 充电座是否贴墙放置；
  2. 充电片是否有污渍；
  3. 地图是否异常；
  4. 电量是否低于回充阈值。
- 客服不得承诺“所有户型都能一次建图成功”。

## 2.5 智能门锁
- 适用场景：符合安装条件的中国大陆住宅木门或金属门。
- 标准包装通常包含：前后面板、锁体、配件包、应急钥匙、说明书。
- 如果用户反馈“无法安装”，则先确认：
  1. 门厚是否在商品页标注范围内；
  2. 门型是否适配；
  3. 开门方向是否与下单选项一致。
- 客服不得承诺“所有门型都免改孔安装”。

# 3. 保修期

## 3.1 整机保修期
- 智能净化器主机：24个月。
- 降噪耳机主机：12个月。
- 胶囊咖啡机主机：18个月。
- 扫地机器人主机：24个月。
- 智能门锁主机：24个月。

## 3.2 主要配件保修期
- 扫地机器人电池：12个月。
- 智能门锁电子配件模块：12个月。
- 咖啡机水泵与电路模块：18个月，随主机保修，不单独延长。
- 耳机充电盒：12个月，前提是与原订单绑定。

## 3.3 耗材与易损件
- 净化器滤芯、咖啡胶囊、扫地机尘袋、边刷、抹布属于耗材或易损件。
- 如果耗材已拆封或已使用，则不适用常规保修。
- 如果耗材未拆封且签收次日起7个自然日内申请退货，则可按无理由退货规则处理。
- 如果耗材到货即破损、漏液、缺件或型号错误，则按质量问题或错发处理。

# 4. 耗材/配件规则

## 4.1 单独购买配件
- 如果用户单独购买配件，则以该配件订单为售后判断依据。
- 如果配件未绑定对应主机型号，客服不得承诺一定兼容。
- 如果用户购买前未核对型号，客服应先引导核对商品页适配清单。

## 4.2 耗材退换
- 如果耗材未拆封且外包装完整，则可在签收次日起7个自然日内申请无理由退货。
- 如果耗材已拆封，但存在漏液、断裂、明显制造瑕疵，则可按质量问题流程处理。
- 如果耗材因用户误购型号不符，且已拆封，则不支持无理由退货。

## 4.3 配件缺失
- 如果整机到货后发现缺少配件，用户应在签收后24小时内提交照片和缺件说明。
- 如果核实缺件成立，则优先补发缺失配件；如果配件无法补发，则按售后工单进一步处理。

# 5. 常见故障处理建议

## 5.1 通用自检原则
- 如果商品无法开机，则优先确认电源、安装状态、按键操作是否正确。
- 如果商品联网失败，则优先确认：
  1. 网络是否为2.4GHz；
  2. App是否为最新版本；
  3. 蓝牙和定位权限是否已开启。
- 如果商品提示异常代码，则应记录异常代码后再进入售后判断。
- 如果商品出现异味、过热、冒烟、异常噪音，则应立即停止使用，并按高优先级售后处理。

## 5.2 净化器
- 滤芯未安装到位可能导致无法启动。
- 长时间未清洁进风口可能导致风量下降。
- 滤芯寿命提示不等于故障提示。

## 5.3 耳机
- 单侧无声时，先尝试恢复出厂设置并重新配对。
- 通话杂音时，先确认麦克风孔位是否被遮挡。
- 长时间满音量使用导致的听感下降，不直接判定为质量故障。

## 5.4 咖啡机
- 首次使用未引水可能导致无法出液。
- 非适配胶囊卡住，不直接判定为机器故障。
- 水垢过多导致流速变慢时，应先引导除垢。

## 5.5 扫地机器人
- 地图紊乱可能与移动充电座、频繁手动搬动主机有关。
- 主刷缠绕毛发导致不转，先引导清理后再判断。
- 轻微碰撞痕迹不直接认定为质量问题。

## 5.6 智能门锁
- 低电量可能导致指纹识别变慢或不上锁。
- 门框偏移可能导致反锁失败。
- 非原装电池、受潮、暴力开门造成的问题，不直接认定为质量问题。

# 6. 不承诺事项

- 不承诺未经检测即可直接认定为质量问题。
- 不承诺所有商品都支持“只退款不退货”。
- 不承诺所有门锁都无需改孔安装。
- 不承诺耳机在所有场景下都达到同等降噪效果。
- 不承诺净化器滤芯在固定天数内一定耗尽或一定耐用。
- 不承诺扫地机器人对所有地毯、门槛、复杂户型均表现一致。
- 不承诺未经仓库、物流、售后中心确认的补发、换新、退款结果。
- 不承诺超出保修期后仍可免费维修。
