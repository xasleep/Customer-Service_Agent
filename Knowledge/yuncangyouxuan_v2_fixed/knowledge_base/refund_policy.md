# 1. 7天无理由退货

## 1.1 适用条件
- 如果商品为实物商品，且不属于本政策列明的不支持情形，则支持7天无理由退货。
- 7天起算时间为用户签收次日00:00开始，至第7个自然日23:59结束。
- 如果用户申请无理由退货，则商品需同时满足：
  1. 主商品、配件、赠品齐全；
  2. 外包装、说明书、标签完整；
  3. 未出现影响二次销售的使用痕迹；
  4. 未发生激活、绑定、安装、刻字或个性化处理。

## 1.2 二次销售判断
- 如果耳机耳罩、门锁面板、净化器机身、咖啡机出液口存在明显使用痕迹，则视为影响二次销售。
- 如果商品缺少原装包装、核心配件或序列号标签，则可拒绝无理由退货申请。
- 如果用户仅拆箱检查但未使用，且商品完整，则不当然视为影响二次销售。

# 2. 不支持无理由退货的情况

- 会员权益类虚拟商品一经到账，不支持无理由退货。
- 已拆封并使用的滤芯、咖啡胶囊、尘袋、抹布等耗材，不支持无理由退货。
- 已安装或已激活的智能门锁，不支持无理由退货。
- 已绑定账号的智能设备，如因绑定导致无法恢复为初始状态，则不支持无理由退货。
- 因用户原因造成的划伤、磕碰、进液、污损，不支持无理由退货。
- 明示为清仓样机、瑕疵特价、定制配件的商品，不支持无理由退货。
- 超过7天无理由退货时限的申请，不支持按无理由退货处理。

# 3. 质量问题退款

## 3.1 质量问题受理时限
- 如果商品在签收次日起7个自然日内出现经核实的质量问题，则用户可申请退货退款或换货。
- 如果商品在签收次日起第8至15个自然日内出现经核实的质量问题，则可申请换货或维修；如果同款无货，则可改为退款。
- 如果商品在15个自然日后出现质量问题，则按保修维修规则处理。

## 3.2 质量问题判定
- 质量问题需以远程排查结果、照片视频、日志信息或售后检测结论为依据。
- 如果商品无法开机、功能失效、核心部件异常，且排除安装或使用问题后仍异常，则可进入质量问题判断。
- 如果仅为外观轻微差异、主观噪音感受、非核心功能体验差异，则不直接认定为质量问题。

## 3.3 到货即损/错发/缺件
- 如果商品到货即破损、错发、缺件，且用户在签收后24小时内提交凭证，则按质量问题或物流责任处理。
- 此类情形可优先安排补发、换货或退款，具体以核实结果为准。

# 4. 退款到账时间

## 4.1 退款处理时效
- 如果退款申请无需退货，且审核通过，则应在24小时内发起退款。
- 如果退款申请需要退货，则以仓库签收并完成验货后24小时内发起退款。
- 如果质量问题需检测，则以检测结论完成后24小时内发起退款。

## 4.2 到账时效
- 余额类支付：通常1至3个工作日到账。
- 第三方支付：通常1至5个工作日到账。
- 支付卡支付：通常3至7个工作日到账。
- 如果超过上述时效仍未到账，客服应创建退款跟踪工单。

# 5. 退货运费承担

## 5.1 无理由退货
- 如果用户因个人原因申请7天无理由退货，则退货运费由用户承担。
- 如果订单下单时产生偏远地区附加运费，则该附加运费不退回。

## 5.2 质量问题退货
- 如果经核实为质量问题、错发、缺件或运输破损，则退货运费由云仓优选承担。
- 如果客服已提供指定取件方式，则用户应优先使用指定方式；未经确认自行寄回产生的超额运费，平台不保证全额承担。

## 5.3 拒收场景
- 如果用户因个人原因拒收非质量问题商品，则往返运费由用户承担。
- 如果因外包装严重破损、明显错发而拒收，则按异常物流或错发处理，用户无需承担责任。

# 6. 换货规则

## 6.1 换货时限
- 如果商品在签收次日起15个自然日内出现经核实的质量问题，则可申请换货。
- 如果超过15个自然日，则原则上不支持直接换货，转为维修处理。

## 6.2 换货范围
- 换货仅支持同款、同颜色、同规格换货。
- 如果原商品无货，则可由用户选择等待补货、改为退款，或在客服确认后更换同价位在售同型号替代款。
- 不支持通过“换货”实现升级型号、跨品类更换。

## 6.3 换货次数
- 同一订单商品在签收后15个自然日内，原则上仅支持一次直接换货。
- 如果换货后再次出现相同核心故障，则需进入售后检测流程重新判断。

# 7. 仅退款规则

## 7.1 可支持仅退款的情形
- 如果订单未发货，用户可申请取消订单并全额退款。
- 如果物流核查确认丢件，则用户可申请仅退款。
- 如果商品存在缺少非核心配件、外观轻微瑕疵且用户愿意保留商品，经审核后可申请部分退款。
- 部分退款金额原则上不超过该商品实付金额的15%。

## 7.2 不支持仅退款的情形
- 如果商品已签收且用户不愿退回商品，仅以“体验不佳”为由要求全额退款，则不支持。
- 如果商品存在质量争议但尚未完成核实，则不得先承诺仅退款。
- 如果商品属于需返仓检测品类，原则上应退回检测后再判断退款。

# 8. 订单取消规则

## 8.1 发货前取消
- 如果订单状态为“待支付”，订单将在15分钟未支付后自动关闭。
- 如果订单状态为“已支付待审核”或“待出库”，用户可申请取消，审核通过后全额退款。
- 如果订单包含预售商品且页面写明“锁单后不可取消”，则以页面说明为准。

## 8.2 发货后取消
- 如果订单已出库但未揽收，可尝试拦截；拦截成功后退款。
- 如果订单已进入运输中，则不支持直接取消，需等待拒收退回或签收后按退货流程处理。
