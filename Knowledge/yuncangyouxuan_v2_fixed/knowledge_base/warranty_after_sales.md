# 1. 售后检测流程

## 1.1 受理前确认
- 如果用户申请售后，客服应先确认：
  1. 订单号；
  2. 商品名称与型号；
  3. 故障现象；
  4. 故障发生时间；
  5. 是否已做基础排查。
- 如果用户无法提供订单号，则可用收货手机号后4位、收件人姓名、下单渠道辅助定位。
- 客服不得在聊天中要求用户提供完整身份证号、完整支付卡号或验证码。

## 1.2 远程初检
- 如果故障可通过远程排查确认，则优先远程初检。
- 远程初检可要求用户提供：
  1. 故障视频；
  2. 异常提示照片；
  3. 序列号照片；
  4. 使用环境说明。
- 如果远程初检已能确认非质量原因，则客服应说明原因并给出处理建议。

## 1.3 返仓检测
- 如果远程初检无法定性，或商品属于需检测品类，则进入返仓检测。
- 返仓检测以售后中心签收商品次日起3个工作日内出具结论为目标时效。
- 如果遇到零件拆检、功能复现困难，则检测时效可延长至5个工作日，需在工单中备注。

# 2. 维修/换货/退款判断

## 2.1 7日内
- 如果商品在签收次日起7个自然日内，经核实为质量问题，则用户可选择退货退款或换货。
- 如果商品存在安全风险，如异常冒烟、过热、漏电迹象，则优先升级高优先级工单，不直接承诺退款结果。

## 2.2 8至15日
- 如果商品在签收次日起第8至15个自然日内，经核实为质量问题，则可换货或维修。
- 如果同款无货，则可改为退款。

## 2.3 16日及保修期内
- 如果商品超过15个自然日但仍在保修期内，则原则上按维修处理。
- 如果同一核心故障已完成2次维修，且第二次维修完成后30个自然日内再次出现同一核心故障，则可申请特殊换货复核。

## 2.4 不予保修但可付费处理
- 如果属于人为损坏、受潮、跌落、私拆、使用非适配电源或非适配耗材造成的故障，则不适用免费保修。
- 如果用户接受付费维修报价，则进入付费维修流程。

# 3. 保修凭证

## 3.1 有效凭证
- 云仓优选平台订单记录可作为默认保修凭证。
- 如果为赠礼转送，需提供原订单号、商品序列号和收件信息以便核验。
- 如果商品序列号缺失、涂改或与订单不匹配，则需人工复核。

## 3.2 保修起算
- 整机保修期自签收次日开始计算。
- 换货后的新机保修期按原订单剩余保修期计算；如果剩余保修期不足90个自然日，则按90个自然日计算。
- 单独补发配件的保修期按原主订单或配件订单规则执行，不额外延长整机保修期。

# 4. 人为损坏

## 4.1 典型人为损坏情形
- 跌落、挤压、磕碰导致外壳裂损或内部结构损坏。
- 进液、受潮、腐蚀。
- 擅自拆机、私自改线、非授权维修。
- 使用非适配耗材导致堵塞、损坏。
- 使用非原装或不符合要求的电源适配器导致故障。
- 智能门锁因暴力开门、门体形变导致的异常。

## 4.2 人为损坏处理
- 如果检测为人为损坏，则不支持免费维修、换货或退款。
- 如果零件可修复，则可提供一次付费维修报价。
- 如果无法维修，则可退回原机，不提供替代补偿承诺。

# 5. 超保处理

## 5.1 超保判定
- 如果商品已超过对应保修期，则按超保处理。
- 超保后不支持免费维修、免费换货或免费退货。

## 5.2 付费维修流程
- 售后中心可提供一次付费维修报价。
- 报价单自发出之日起7个自然日内有效。
- 如果用户在7个自然日内未确认，则默认放弃维修，商品按“退回未修”处理。

## 5.3 退回未修
- 如果用户放弃付费维修，则可选择寄回。
- 如果用户在收到“可退回”通知后30个自然日内未确认收件信息，则工单将进入二次提醒。
- 如果二次提醒后仍无回复，则商品继续保管30个自然日；超期后的进一步处理以工单通知为准，客服不得口头承诺无限期保管。

# 6. 售后工单优先级

## 6.1 P1 紧急
- 商品出现冒烟、异常发热、疑似漏电、门锁无法从室内正常开启等安全风险。
- 目标响应时效：30分钟内转人工。
- 目标处理动作：先停用提醒，再创建高优先级工单。

## 6.2 P2 高
- 签收15日内整机无法正常使用。
- 物流签收争议且用户明确表示未收到高价值商品。
- 目标响应时效：2小时内人工接手。

## 6.3 P3 中
- 配件缺失、非核心功能异常、重复咨询未解决。
- 目标响应时效：24小时内跟进。

## 6.4 P4 低
- 使用指导、保养咨询、兼容性咨询。
- 目标响应时效：48小时内处理。

# 7. 售后结论表达规范

- 如果尚未检测，不得直接表述为“确定是质量问题”。
- 如果仓库未签收，不得直接表述为“已经退款”。
- 如果维修中心未出结论，不得直接承诺“必换新”。
- 应使用以下表达：
  - “已为您提交检测，最终以检测结果为准。”
  - “先帮您核对订单和故障信息，再给您对应处理路径。”
  - “当前可以先按售后流程登记，结果需要以复核完成为准。”
